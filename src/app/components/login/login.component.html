JavaFX Control de Alumnos

El objetivo de este proyecto es llevar el control de los alumnos y se desempeñan varias funciones como agregar un nuevo alumno, asignar horario, ver el salón, listar las carreras técnicas, entre otras, estas funciones se pueden crear, editar, eliminar y crear reporte del estado actual, este proyecto estuvo en desarrollo en un tiempo estimado de 3 meses.
